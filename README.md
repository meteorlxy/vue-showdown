## Usage

## Contributing

Feel free to submit Issues and PRs if you meet any problems or have any ideas.

### Develop

```sh
# Clone this repo
git clone https://github.com/meteorlxy/vue-showdown
cd vue-showdown

# Install dependencies
npm i

# Start the dev-server
npm run dev
```
